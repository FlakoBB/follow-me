# FollowMe

Página de enlaces de perfil personal tipo "link in bio", construida con Vanilla JS y Vite. Muestra redes sociales con íconos SVG y un botón para compartir la página.

## Demo

[https://flakobb.github.io/follow-me](https://flakobb.github.io/follow-me)

## Stack

- Vanilla JavaScript (ES Modules)
- Vite
- SCSS
- StandardJS

## Agregar un enlace

Edita `src/data/links.json`:

```json
{
  "name": "Mi Red",
  "link": "https://...",
  "icon": "MiIconoIcon"
}
```
